## DSP

`DSP/` 保存数字信号处理课程实验材料。每个实验目录通常包含：

- `lab*.md`：实验报告源文件；
- `lab*.pdf`：导出的实验报告；
- `image*.png` / 其他图片：报告中使用的实验结果图；
- MATLAB 脚本或相关附件。

目前包含 Lab 1 到 Lab 4，主题包括 LTI 系统响应、卷积、均衡器设计、陷波器设计、谱分析、窗函数影响、IIR/FIR 滤波器分析等。

## gkc2

`gkc2/` 保存工科创项目材料，重点是家庭负荷、光伏、EV 负荷和储能策略的建模与仿真。

主要内容：

- `代码/`：MATLAB 程序，包括主程序、数据准备、储能仿真和结果绘图；
- `csv文件/`：年度负荷曲线、光伏曲线、EV 曲线以及策略扫描结果；
- `工科创项目报告.docx`：项目报告；
- `README.txt`：更详细的运行说明和文件解释。

这部分内容更像一个小型课程项目，关注数据生成、策略比较和结果可视化。

## 嵌入式

`嵌入式/` 保存嵌入式系统课程大作业相关材料，主题为 S800 智能联网时钟。

主要内容：

- `docs/`：项目简介和报告文档；
- `mcu/`：开发板端工程、驱动库和嵌入式代码；
- `pc_host/`：上位机程序；
- `README.md`：项目功能、运行方式和验收测试说明。

这个项目包含板端显示、串口通信、网络对时、天气获取、自动昼夜模式、数据可视化和蜂鸣器音乐等功能，是本仓库中工程完整度最高的一部分。

## How To Read This Repository

如果只是快速了解学习内容，可以按这个顺序查看：

1. 先看本 README，了解本学期的课程和项目分布；
2. 查看 `DSP/README.md`，了解数字信号处理实验；
3. 查看 `gkc2/README.txt`，了解 MATLAB 仿真项目；
4. 查看 `嵌入式/README.md` 和 `嵌入式/docs/`，了解嵌入式大作业；
5. 需要复现实验或项目时，再进入对应代码目录。
